# LexiNote — Dyslexia-Friendly Learning Platform.

## Executive Summary

LexiNote is an accessibility-focused application designed to enhance reading and writing experiences for individuals with dyslexia. The system transforms standard text into a more readable format by improving spacing, structure, and visual clarity, thereby reducing cognitive load and visual stress.

In addition to text formatting, LexiNote integrates supportive features such as text-to-speech, writing assistance, error detection, and speech correction. These capabilities aim to assist users in both consuming and producing content more effectively.

The application is built with a focus on simplicity, usability, and inclusivity. By combining fundamental text processing techniques with assistive functionalities, LexiNote addresses real-world challenges faced by dyslexic users and promotes more accessible digital interaction.

Overall, LexiNote demonstrates the application of technology in solving accessibility problems, with potential for further expansion through advanced natural language processing and user-centric design enhancements.
## Objectives
- Enhance readability of digital text for dyslexic users
- Reduce letter crowding and visual confusion
- Provide a simple interface for real-time text conversion

## Features
- Simple user interface
- Text Conversion: Transforms standard text into a dyslexia-friendly format with improved spacing and readability.
- Writing Assistance: Provides guided writing support to help users form clear and structured text.
- Text-to-Speech: Converts written text into audio output to assist users with reading difficulties.
- Notes Conversion: Converts raw or unstructured text into organized and readable notes.
- Error Detection: Identifies common spelling and structural errors in text.
- Speech Correction: Improves spoken input by refining clarity and correcting recognized speech errors.

## Problem Statement
Readers with dyslexia often experience:
- Difficulty distinguishing similar characters
- Visual crowding of letters and words
- Increased effort in reading dense text

LexiNote addresses these challenges by restructuring text presentation to improve clarity and readability.
